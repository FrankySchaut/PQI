# Relevance Audit Protocol — PQI v0.2
Purpose: ensure inputs remain non-personal and ethically bounded.
Domains: Semantic (no identifiers), Structural (abstract vectors), Ethical (proportional use).
Procedure: pre-entry scrub → source declaration → random 10% sample → reflection clause.
Acceptable: synthetic, derived, non-reversible embeddings. Disallowed: raw human text/media.
Verification rule: assert all 0≤x≤1.
